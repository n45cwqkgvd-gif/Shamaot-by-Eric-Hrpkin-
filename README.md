# 🚗 מערכת שמאות רכב — חוות דעת מקצועית

מערכת ווב מקצועית לכתיבת שומות ודוחות שמאי רכב, הכוללת תחשיב אוטומטי, סכמה ויזואלית של הרכב ודוח הדפסה מלא.

---

## ✅ תכונות המערכת

- **3 סוגי שומה** — לתיקון / אובדן להלכה / אובדן כללי (לפי תקנות משרד התחבורה)
- **סכמה ויזואלית** — לחץ על אזור הפגיעה, בחר חלקים ומחירים
- **חישוב אוטומטי** — חלקים + עבודות + צביעה + מע"מ
- **מד נזק חי** — מציג אחוז נזק בזמן אמת ומציע את סוג השומה הנכון
- **פרופיל שמאי** — מוגדר פעם אחת, מופיע בכל השומות
- **דוח הדפסה** — פורמט מקצועי מלא כולל כותרת, טבלאות, הצהרה משפטית וחתימה
- **אחסון מקומי** — כל השומות נשמרות ב-localStorage בדפדפן

---

## 🚀 העלאה ל-GitHub Pages (5 דקות)

### שלב 1 — צור repository חדש
1. כנס ל-[github.com](https://github.com) והתחבר לחשבון
2. לחץ על **"New"** (כפתור ירוק)
3. תן שם לפרויקט, לדוגמה: `shamay-rechev`
4. בחר **Public**
5. לחץ **"Create repository"**

### שלב 2 — העלה את הקובץ
1. בדף ה-repository החדש לחץ על **"uploading an existing file"**
2. גרור את הקובץ `index.html` לתוך הדפדפן
3. כתוב בתיבת ה-commit: `הוספת מערכת שמאות`
4. לחץ **"Commit changes"**

### שלב 3 — הפעל GitHub Pages
1. לחץ על **Settings** (בתפריט העליון של ה-repository)
2. בסרגל השמאלי לחץ על **Pages**
3. תחת **"Branch"** בחר: `main` ולחץ **Save**
4. המתן כ-2 דקות

### שלב 4 — קבל את הקישור
הכתובת תהיה:
```
https://[שם-המשתמש-שלך].github.io/shamay-rechev/
```

## 👥 שיתוף עם תלמידים

כל תלמיד שנכנס לאתר:
- **רואה מסך ריק** — האתר הוא כלי, לא מאגר משותף
- **מגדיר פרופיל שמאי אישי** שלו (נשמר בדפדפן שלו)
- **יוצר שומות** שנשמרות רק אצלו בדפדפן
- **מדפיס / שומר PDF** מהאתר

> ⚠️ **חשוב:** השומות נשמרות ב-localStorage של הדפדפן בלבד.
> אין שיתוף אוטומטי בין משתמשים.
> לשיתוף — כל אחד מדפיס PDF בעצמו.

## 🖨️ הדפסה ו-PDF

1. לחץ על כפתור **"הדפסה"** בשומה
2. הדפדפן פותח תצוגת הדפסה
3. לשמירה כ-PDF: בחר **"Save as PDF"** / **"שמור כ-PDF"**
